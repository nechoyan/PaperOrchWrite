# Example: Agentic Security Industry Report

A fully-executed PaperOrchestra run producing a 22-page technical white paper on AI coding agent security. This example demonstrates the pipeline on a **real-world industry research report** (not a traditional academic experiment paper), showing that PaperOrchestra generalises beyond CS conference submissions.

**Output:** [`workspace/final/paper.pdf`](workspace/final/paper.pdf) — 22 pages, compiled clean with zero LaTeX errors.

## Pipeline Execution

All five PaperOrchestra agents ran in sequence (Steps 2 and 3 in parallel):

| Step | Agent | Output | Notes |
|------|-------|--------|-------|
| 1 | Outline Agent | `workspace/outline.json` | 6 figures, 4 related-work clusters, 10 sections, 78 citation hints. Validated with `validate_outline.py`. |
| 2 | Plotting Agent | `workspace/figures/*.png` + `captions.json` | 7 figures at 300 DPI using matplotlib. All data from `experimental_log.md`. |
| 3 | Literature Review Agent | `workspace/drafts/intro_relwork.tex` + `workspace/refs.bib` | 491 lines of LaTeX, 22 BibTeX entries, 100% citation coverage gate. |
| 4 | Section Writing Agent | `workspace/drafts/paper.tex` | 1,499 lines. Both gates (orphan-cite, latex-sanity) passed. |
| 5 | Content Refinement Agent | `workspace/final/paper.tex` | 3 iterations. Final accepted snapshot compiled to `paper.pdf`. |

## What Makes This Example Interesting

**1. Non-standard document type.** The `conference_guidelines.md` specifies a single-column industry white paper, not a two-column ACM/IEEE paper. The pipeline adapted: single-column layout, 11pt article class, professional header/footer, booktabs tables throughout.

**2. Data-heavy experimental log.** The `experimental_log.md` contains 11 structured data tables (market figures, attack success rates, competitive capability matrix, vendor funding timeline, incident catalogue, research paper list). The Plotting Agent generated 6 figures directly from this tabular data with no pre-existing figures provided.

**3. Competitive analysis as a section type.** The outline placed a full competitive analysis section (vendor profiles + 12×7 capability matrix heatmap) — not a typical academic section. The pipeline handled it without modification.

**4. Refinement agent added a figure.** During iter1, the Content Refinement Agent identified that attack success rates deserved a dedicated figure and generated `fig_attack_success_rates.png` — demonstrating that the refinement loop can extend the plotting output, not just edit text.

## Reproducing This Example

```bash
# Clone PaperOrchestra
git clone https://github.com/Ar9av/PaperOrchestra.git ~/paper-orchestra
cd ~/paper-orchestra
pip install -r requirements.txt

# Copy this example's inputs to a fresh workspace
python skills/paper-orchestra/scripts/init_workspace.py --out /tmp/agentic-security-ws
cp examples/agentic-security-report/inputs/* /tmp/agentic-security-ws/inputs/

# Validate inputs
python skills/paper-orchestra/scripts/validate_inputs.py --workspace /tmp/agentic-security-ws

# Run the pipeline (tell your agent):
# "Run the paper-orchestra skill on workspace at /tmp/agentic-security-ws"
```

Requires: Python 3.10+, pdflatex (texlive-latex-recommended + texlive-latex-extra), matplotlib, numpy.
